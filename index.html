Site sobre mim

O projeto é uma página web criada com HTML e CSS, destinada a falar sobre mim.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o ntexto com uma fonte importada do Google Fonts e aplicando um esquema de cores e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária.

Alura Books

O projeto consiste em uma biblioteca online, desenvolvida com HTML e CSS. O objetivo é criar uma plataforma que facilite o acesso a livros variados.

O HTML estrutura o conteúdo da página. O CSS é utilizado para estilizar a página, empregando um esquema de cores e layout responsivo para melhorar a experiência do usuário.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido pela coleção de livros. Tem um layout flexível que se adapta a diferentes tamanhos de tela.
